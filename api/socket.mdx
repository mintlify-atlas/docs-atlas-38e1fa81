## makeWASocket

The primary function to create a WhatsApp Web socket connection.

```typescript
function makeWASocket(config: UserFacingSocketConfig): WASocket
```

<ParamField path="config" type="UserFacingSocketConfig" required>
  Configuration object for the socket connection. See [Socket Configuration](/api/socket-config) for details.
</ParamField>

<ResponseField name="WASocket" type="object">
  Returns a socket instance with all methods and properties for interacting with WhatsApp Web API.
</ResponseField>

### Example

```typescript
import makeWASocket from '@whiskeysockets/baileys'
import { useMultiFileAuthState } from '@whiskeysockets/baileys'

const { state, saveCreds } = await useMultiFileAuthState('auth_info')

const sock = makeWASocket({
    auth: state
})

sock.ev.on('connection.update', (update) => {
    const { connection, lastDisconnect, qr } = update
    
    if(qr) {
        console.log('QR Code:', qr)
    }
    
    if(connection === 'close') {
        console.log('Connection closed')
    } else if(connection === 'open') {
        console.log('Connection opened')
    }
})

sock.ev.on('creds.update', saveCreds)
```

## WASocket Methods

The `WASocket` object returned by `makeWASocket` includes the following categories of methods:

### Core Properties

<ResponseField name="ev" type="BaileysEventEmitter">
  Event emitter for listening to WhatsApp events. See [Events](/api/events) for available events.
</ResponseField>

<ResponseField name="user" type="Contact | undefined">
  Information about the authenticated user.
</ResponseField>

<ResponseField name="ws" type="WebSocket">
  The underlying WebSocket connection.
</ResponseField>

<ResponseField name="authState" type="AuthenticationState">
  Current authentication state including credentials and keys.
</ResponseField>

### Messaging Methods

<ResponseField name="sendMessage" type="function">
  ```typescript
  sendMessage(
    jid: string,
    content: AnyMessageContent,
    options?: MessageRelayOptions
  ): Promise<proto.WebMessageInfo>
  ```
  
  Send a message to a chat.
  
  <ParamField path="jid" type="string" required>
    The JID (WhatsApp ID) of the recipient (e.g., "1234567890@s.whatsapp.net" for individual, "1234567890@g.us" for group)
  </ParamField>
  
  <ParamField path="content" type="AnyMessageContent" required>
    Message content (text, image, video, document, etc.)
  </ParamField>
  
  <ParamField path="options" type="MessageRelayOptions">
    Additional options for message sending (quoted message, etc.)
  </ParamField>
</ResponseField>

<ResponseField name="sendReceipt" type="function">
  ```typescript
  sendReceipt(
    jid: string,
    participant: string | undefined,
    messageIds: string[],
    type: MessageReceiptType
  ): Promise<void>
  ```
  
  Send read/delivery receipts for messages.
</ResponseField>

<ResponseField name="sendPresenceUpdate" type="function">
  ```typescript
  sendPresenceUpdate(type: WAPresence, toJid?: string): Promise<void>
  ```
  
  Update presence status (typing, recording, online, etc.).
  
  <ParamField path="type" type="WAPresence" required>
    Presence type: `'unavailable'` | `'available'` | `'composing'` | `'recording'` | `'paused'`
  </ParamField>
  
  <ParamField path="toJid" type="string">
    Target JID for the presence update (optional, defaults to general presence)
  </ParamField>
</ResponseField>

<ResponseField name="readMessages" type="function">
  ```typescript
  readMessages(keys: WAMessageKey[]): Promise<void>
  ```
  
  Mark messages as read.
</ResponseField>
