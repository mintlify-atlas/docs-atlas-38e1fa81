## AnyMessageContent

Union type representing all possible message content types that can be sent.

```typescript
type AnyMessageContent =
  | AnyRegularMessageContent
  | { forward: WAMessage; force?: boolean }
  | { delete: WAMessageKey }
  | { disappearingMessagesInChat: boolean | number }
  | { limitSharing: boolean }
```

## AnyRegularMessageContent

Union type for standard message content types.

```typescript
type AnyRegularMessageContent = (
  | TextMessageContent
  | AnyMediaMessageContent
  | { event: EventMessageOptions }
  | PollMessageContent
  | ContactsMessageContent
  | LocationMessageContent
  | { react: proto.Message.IReactionMessage }
  | ButtonReplyMessageContent
  | GroupInviteMessageContent
  | ListReplyMessageContent
  | PinMessageContent
  | ProductMessageContent
  | SharePhoneNumber
  | RequestPhoneNumber
) & ViewOnce
```

## Text Messages

### TextMessageContent

Send text messages with optional mentions, link previews, and context.

<ParamField body="text" type="string" required>
  The text content of the message
</ParamField>

<ParamField body="linkPreview" type="WAUrlInfo | null">
  Optional link preview information for URLs in the text
</ParamField>

<ParamField body="mentions" type="string[]">
  List of JIDs that are mentioned in the text (e.g., `['1234567890@s.whatsapp.net']`)
</ParamField>

<ParamField body="contextInfo" type="proto.IContextInfo">
  Additional context information (quoted messages, forwarding info, etc.)
</ParamField>

<ParamField body="edit" type="WAMessageKey">
  Message key of the message to edit (for editing existing messages)
</ParamField>

**Type Definition:**
```typescript
type TextMessageContent = {
  text: string
  linkPreview?: WAUrlInfo | null
} & Mentionable & Contextable & Editable
```

## Media Messages

```typescript
type AnyMediaMessageContent = (
```

### ImageMessageContent

Send image messages with optional captions and dimensions.

<ParamField body="image" type="WAMediaUpload" required>
  The image file as Buffer, stream, or URL
</ParamField>

<ParamField body="caption" type="string">
  Optional caption for the image
</ParamField>

<ParamField body="jpegThumbnail" type="string">
  JPEG thumbnail as base64 string
</ParamField>

<ParamField body="width" type="number">
  Image width in pixels
</ParamField>

<ParamField body="height" type="number">
  Image height in pixels
</ParamField>

<ParamField body="mentions" type="string[]">
  List of mentioned JIDs in the caption
</ParamField>

<ParamField body="contextInfo" type="proto.IContextInfo">
  Additional context information
</ParamField>

<ParamField body="mimetype" type="string">
  MIME type of the image (e.g., 'image/jpeg', 'image/png')
</ParamField>

**Type Definition:**
```typescript
type ImageMessageContent = {
  image: WAMediaUpload
  caption?: string
  jpegThumbnail?: string
} & Mentionable & Contextable & WithDimensions
```

<ParamField body="width" type="number">

</ParamField>

<ParamField body="height" type="number">
  Video height in pixels
</ParamField>

<ParamField body="mentions" type="string[]">
  List of mentioned JIDs in the caption
</ParamField>

<ParamField body="contextInfo" type="proto.IContextInfo">
  Additional context information
</ParamField>

<ParamField body="mimetype" type="string">

</ParamField>

## Interactive Messages

### PollMessageContent

Create poll messages with multiple selectable options.

<ParamField body="poll" type="PollMessageOptions" required>
  Poll configuration object
</ParamField>

<ParamField body="poll.name" type="string" required>
  The poll question
</ParamField>

<ParamField body="poll.values" type="string[]" required>
  Array of poll options
</ParamField>

<ParamField body="poll.selectableCount" type="number">
  Number of options users can select (default: 1)
</ParamField>

<ParamField body="poll.messageSecret" type="Uint8Array">
  32-byte message secret to encrypt poll selections
</ParamField>

<ParamField body="poll.toAnnouncementGroup" type="boolean">
  Whether the poll is sent to an announcement group
</ParamField>

<ParamField body="mentions" type="string[]">
  List of mentioned JIDs
</ParamField>

<ParamField body="contextInfo" type="proto.IContextInfo">
  Additional context information
</ParamField>

**Type Definition:**
```typescript
type PollMessageContent = {
  poll: PollMessageOptions
} & Mentionable & Contextable & Editable

type PollMessageOptions = {
  name: string
  selectableCount?: number
  values: string[]
  messageSecret?: Uint8Array
  toAnnouncementGroup?: boolean
}
```

### EventMessageContent

Create event messages for scheduling meetings and calls.

<ParamField body="event" type="EventMessageOptions" required>
  Event configuration object
</ParamField>

<ParamField body="event.name" type="string" required>
  Event name/title
</ParamField>

<ParamField body="event.startDate" type="Date" required>
  Event start date and time
</ParamField>

<ParamField body="event.description" type="string">
  Optional event description
</ParamField>

<ParamField body="event.endDate" type="Date">
  Optional event end date and time
</ParamField>

<ParamField body="event.location" type="WALocationMessage">
  Optional event location
</ParamField>

<ParamField body="event.call" type="'audio' | 'video'">
  Type of call associated with the event
</ParamField>

<ParamField body="event.isCancelled" type="boolean">
  Whether the event is cancelled
</ParamField>

<ParamField body="event.isScheduleCall" type="boolean">
  Whether this is a scheduled call
</ParamField>

<ParamField body="event.extraGuestsAllowed" type="boolean">
  Whether extra guests are allowed
</ParamField>

<ParamField body="event.messageSecret" type="Uint8Array">
  Message secret for encryption
</ParamField>

**Type Definition:**
```typescript
type EventMessageOptions = {
  name: string
  description?: string
  startDate: Date
  endDate?: Date
  location?: WALocationMessage
  call?: 'audio' | 'video'
  isCancelled?: boolean
  isScheduleCall?: boolean
  extraGuestsAllowed?: boolean
  messageSecret?: Uint8Array
}
```

## Message Actions

### ForwardMessage

Forward existing messages.

<ParamField body="forward" type="WAMessage" required>
  The message to forward
</ParamField>

<ParamField body="force" type="boolean">
  Force forwarding even if message is not forwardable
</ParamField>

**Type Definition:**
```typescript
type ForwardMessage = {
  forward: WAMessage
  force?: boolean
}
```

### DeleteMessage

Delete messages (your own or anyone's in a group if you're an admin).

<ParamField body="delete" type="WAMessageKey" required>
  Message key of the message to delete
</ParamField>

**Type Definition:**
```typescript
type DeleteMessage = {
  delete: WAMessageKey
}
```

### PinMessage

Pin messages in a chat.

<ParamField body="pin" type="WAMessageKey" required>
  Message key of the message to pin
</ParamField>

<ParamField body="type" type="proto.PinInChat.Type" required>
  Pin type (PIN_IN_CHAT or UNPIN_IN_CHAT)
</ParamField>

<ParamField body="time" type="86400 | 604800 | 2592000">
  Pin duration: 86400 (24 hours), 604800 (7 days), or 2592000 (30 days)
</ParamField>

**Type Definition:**
```typescript
type PinMessage = {
  pin: WAMessageKey
  type: proto.PinInChat.Type
  time?: 86400 | 604800 | 2592000
}
```

### DisappearingMessagesToggle

Toggle disappearing messages in a chat.

<ParamField body="disappearingMessagesInChat" type="boolean | number" required>
  `false` to disable, or a number (in seconds) to enable with expiration time
</ParamField>

**Type Definition:**
```typescript
type DisappearingMessagesToggle = {
  disappearingMessagesInChat: boolean | number
}
```

### LimitSharingToggle

Limit message forwarding and sharing.

<ParamField body="limitSharing" type="boolean" required>
  Whether to limit sharing of messages
</ParamField>

**Type Definition:**
```typescript
type LimitSharingToggle = {
  limitSharing: boolean
}
```
